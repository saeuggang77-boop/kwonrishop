# 권리샵 브랜드 가이드

_Version 1.0 · 2026-04-15_

상가 직거래 플랫폼 권리샵의 브랜드 방향, 시각 자산, 구현 규칙을 정의합니다.

---

## 1. 브랜드 철학

### 포지셔닝
> "중개수수료 0원, 그게 직거래다."

권리샵은 사장님과 예비창업자를 **직접** 연결합니다. 중개인을 거치지 않을 때 권리금은 가장 **투명하고 정직하게** 거래됩니다.

### 톤 & 무드
- **성숙함** — 과장하지 않고 차분하게 말한다
- **따뜻함** — 자영업의 꿈과 현실을 존중한다
- **신뢰감** — 숫자는 정확하게, 용어는 쉽게
- **실용성** — 기능이 먼저, 장식은 필요한 곳에만

### 지양하는 톤
- 힙하거나 트렌디한 스타트업 톤 (사용자 연령대와 불일치)
- 화려한 그라데이션·네온·3D (저가·잡지 느낌 유발)
- 보수적 파랑·네이비 (경쟁사 포화, 차별성 없음)
- 급박한 빨강·세일 느낌 (권리금 거래와 맞지 않음)

- **파랑·네이비 사용 금지** (경쟁사 포화)

- **순수 흰색(#FFF) 사용 금지** — 항상 `cream` 또는 그 계열
- **2개 이상 강조색 동시 사용 금지** — terra는 한 화면에 1~2건만

## 3. 타이포그래피

### 폰트
| 역할 | 폰트 | 이유 |
|------|------|------|
| 한글 기본 | **Pretendard Variable** | 한국어 최적화, 가독성 우수 |
| 영문/숫자 디스플레이 | **Fraunces** (serif) | 성숙하고 따뜻한 세리프, 권리금 숫자에 무게감 |
| 모노 (거의 사용 안 함) | system-mono | 금액 정렬 필요 시만 |

### 크기 체계
| 레벨 | 크기 | 용도 |
|------|------|------|
| Display | 72~84px | 히어로 메인 카피 |
| H1 | 48~56px | 페이지 타이틀 |
| H2 | 32~40px | 섹션 타이틀 |
| H3 | 22~28px | 카드/모달 타이틀 |
| Body-lg | 18px | 히어로 서브카피 |
| Body | 16px | 기본 본문 |
| Body-sm | 14px | 보조 텍스트 |
| Caption | 13px | 메타정보, 라벨 |
| Micro | 11~12px | 키커, 카테고리 라벨 |

### 타이포 규칙
- 한글은 **Pretendard Bold/SemiBold** 위주
- 영문·숫자 디스플레이는 **Fraunces Italic** 으로 강세
- **가격 표기**는 무조건 Fraunces (예: `1.8억`, `7,500만`)
- 레터 스페이싱: 한글 `-0.02em` (타이트), 영문 키커 `+0.1em` (와이드)
- 한 화면에 세리프 강세는 2곳 이하

## 4. 시그니처 요소

### 로고 / 워드마크
```
● 권리샵
↑ 테라코타 #D96C4F
  ↑ 그린 #1F3F2E, Pretendard ExtraBold, -0.04em
```

- 가로형: `● 권리샵` (기본)
- 심볼만: `●` (파비콘, 앱 아이콘, 로딩 스피너)
- 금지: 외곽선 로고, 기울이기, 색상 변경
- 최소 크기: 가로형 80px, 심볼 16px

### 시그니처 장치
- **테라코타 ●** — 로고, 리스트 구분점, 상태 인디케이터
- **Dashed 구분선** — 가격 행 위(`border-top: 1px dashed var(--line-deep)`)
- **Fraunces Italic 수치** — 권리금·월매출·창업비 등 돈 관련 숫자
- **Pill 버튼** — 기본 버튼은 `border-radius: 99px`
- **카드 그림자** — `0 8px 32px rgba(31, 63, 46, 0.08)` (그린이 녹은 그림자)

## 5. 간격·라운딩·그림자 토큰

### Spacing
Tailwind 기본 사용 (`gap-2`, `p-4`, `space-y-6` 등).
사이트 표준:
- 카드 내부 padding: `p-6` (24px)
- 섹션 세로 간격: `py-20` ~ `py-24` (80~96px)
- 카드 간 gap: `gap-4` ~ `gap-6` (16~24px)

### Radius
| 용도 | 값 | Tailwind |
|------|-----|----------|
| 카드 | 24px | `rounded-3xl` (custom) |
| 입력/폼 | 16px | `rounded-2xl` |
| 버튼 | 99px (pill) | `rounded-full` |
| 배지/칩 | 99px | `rounded-full` |
| 작은 요소 | 8px | `rounded-lg` |

### Shadow
| 토큰 | 값 |
|------|-----|
| `shadow-card` | `0 8px 32px rgba(31, 63, 46, 0.08)` |
| `shadow-card-hover` | `0 16px 40px rgba(31, 63, 46, 0.12)` |
| `shadow-soft` | `0 2px 12px rgba(31, 63, 46, 0.06)` |

## 6. 컴포넌트 규칙

### 버튼
| 타입 | 배경 | 텍스트 | 보더 | 라운딩 |
|------|------|--------|------|--------|
| Primary | `green-700` | `cream` | - | full |
| Secondary | `cream` | `green-700` | `line` | full |
| Accent (CTA) | `terra-500` | `cream` | - | full |
| Ghost | transparent | `green-700` | - | lg |
| Destructive | `red-500` | `white` | - | full |

### 티어 뱃지 (중요)
BASIC/PREMIUM/VIP를 **색으로 명확히 구분**.

| 등급 | 배경 | 텍스트 |
|------|------|--------|
| FREE | `cream-elev` | `muted` |
| BASIC | `green-100` | `green-700` |
| PREMIUM | `green-700` | `cream` |
| VIP / GOLD | `terra-500` | `cream` |

### 카드
- 기본 배경: `cream`
- 보더: `1px solid line`
- 라운딩: `rounded-3xl` (24px)
- VIP 카드: `border: 2px solid green-700` + `bg-cream-elev`
- Hover: `translate-y(-4px)` + `shadow-card-hover`

### 입력 폼
- 배경 `cream`, 보더 `line`
- 포커스: `outline: 2px solid green-700`
- 라운딩 `rounded-2xl`

## 9. 다크모드

> **현 버전 지원하지 않음.**

B안 정체성은 크림 배경에서 완성됩니다. 다크모드는 별도 팔레트 재설계가 필요하며, 초기 런칭 이후 별도 논의합니다.

- 모든 `dark:` 클래스 제거됨
- `prefers-color-scheme: dark` 미디어 쿼리 무시
- ThemeProvider/ThemeToggle 비활성화

## 10. Do / Don't

### Do ✓
- 그린 배경 위에 크림 텍스트 → 시그니처 조합
- 가격은 Fraunces 세리프로 차별화
- 카드는 항상 `rounded-3xl`
- 섹션 여백은 `py-24` 이상 확보
- 테라코타는 한 화면에 한 포인트만

### Don't ✗
- 파랑·네이비 도입 금지
- 이모지 사용 금지
- 순수 #000, #FFF 금지
- 한 카드에 3개 이상 뱃지 금지
- 그라데이션 남용 (hero 1회만 허용)
